# 字体文件目录

这个目录用于存放博客系统使用的自定义字体文件。

## 需要的字体文件

博客需要以下字体文件:
- Inter-Bold.woff
- Inter-Regular.woff

## 获取字体文件

你可以从以下地方下载所需的字体:

1. 官方Inter字体网站: https://rsms.me/inter/
2. Google Fonts: https://fonts.google.com/specimen/Inter

下载后的字体文件应放置在这个目录中。

## 注意

如果你不想使用自定义字体，可以修改 `components/3d/BlogHomeScene.tsx` 文件，从 Text 组件中移除 font 属性，这样系统将使用默认字体。
